# Visual Direction

## Target Look

The scene should read as a believable tropical or humid frontier outpost, not an empty test range.

## Visual Priorities

- Clear dinosaur silhouettes at distance.
- A grounded base with fences, barriers, lights, and layered terrain.
- Depth in the horizon through fog, atmosphere, props, and background structure.
- Weapon and HUD elements that support the scene without dominating it.
- Weak-point readability that feels tactical rather than arcade-like.

## Prefer

- Rich but restrained color separation.
- Wet ground, foliage variation, rocks, structures, and utility props.
- Lighting that creates contrast without washing out the scene.
- Damage and hit feedback that is visible but not dirty or overwhelming.

## Avoid

- Placeholder boxes, flat cylinders, or obviously temporary geometry in the final view.
- Giant empty sky as the main background.
- UI panels that block the aiming area.
- Effects that look like noise instead of impact.
- Art that makes dinosaurs or the base blend into the same value range.
